Potassium — 混音总线处理器

信号链

Input Gain → Compressor → EQ → [Oversampling] Saturator → Stereo Width → Limiter → Output Gain

Gain Staging 使用建议

本插件内置增益分级系统。推荐 Top-Down Mix 工作流：

1. 从 Kick 开始独奏，调整输入增益使峰值电平落在 -6dB 附近

2. 电平数值在 -6dB 左右会亮黄或紫色，以此作为参考

3. 以 Kick 为基准逐轨平衡，最终总线的 RMS 电平控制在 -6dB 附近

4. 甜点区间内插件响应最佳，电平数值变黄/紫色即为达到理想工作点
