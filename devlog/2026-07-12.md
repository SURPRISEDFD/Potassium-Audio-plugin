# 2026-07-12 开发日志 — 原型机完成 🎉

## 最终原型机信号链
```
InputGain → ButterComp2 → Drive(PurestDrive) → SmoothEQ3(3-band) → StereoFX(Wide) → Limiter → OutputGain
```

## 最终参数
| 参数 | 范围 | 默认 |
|------|------|------|
| Input Gain | -12~+12 dB | 0 |
| Comp | 0~0.8 | 0.5 |
| Drive | 0~1 | 0.2 |
| EQ Low | -6~+6 dB | 0 |
| EQ Mid | -6~+6 dB | 0 |
| EQ High | -6~+6 dB | 0 |
| Wide | 0~1 | 0 |
| Limit Gain | 0~+12 dB | 0 |
| Output Gain | -24~+24 dB | 0 |
| Oversampling | Off/2x/4x/8x | 4x |
| Bypass | On/Off | Off |
| Byp Comp/Drive/EQ/Stereo/Limit | On/Off | Off |

## 预设
- Default / 100% / 200% 三个内置槽
- 支持重命名和自动保存

## 算法来源
- Compressor: AirWindows ButterComp2 (MIT)
- Drive: AirWindows PurestDrive (MIT)
- EQ: AirWindows SmoothEQ3 (MIT)
- Stereo: PurestDrive饱和染色Side通道
- Limiter: 自编透明砖墙
